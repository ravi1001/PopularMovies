IMPORTANT INSTRUCTIONS
----------------------

This application queries movie data from the TMDB server
and requires an API key issued by TMDB to perform it's
core functionality. 

Please request an API key from TMDB by first creating
an account here: https://www.themoviedb.org/account/signup.

Once you have the API key please specify this key in the 
res/values/strings.xml file into a string having the name
"tmdb_api_key".
